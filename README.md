# Glow Beauty 💄✨

A fully responsive, aesthetically pleasing e-commerce landing page for a makeup and beauty store called **Glow Beauty**. This project demonstrates front-end web design skills using HTML, CSS, and Vanilla JavaScript.

## 🌟 Features

- **Hero Section**: An eye-catching hero banner with a gradient background, a call-to-action button, and elegantly floating animated images.
- **Product Showcases**: Beautifully designed sections to display top items including "Our Products", "New Arrivals", and "Shop By Category".
- **Scroll Animations**: Smooth reveal animations as you scroll down the page, enhancing the user experience.
- **Responsive Design**: Adapts seamlessly to different screen sizes, giving an optimal viewing experience on both desktop and mobile devices.
- **Image Galleries**: Clean, grid-based layouts to explore various makeup products.
- **Modern UI**: Polished typography using Google's "Poppins" font, subtle drop shadows, and scale hover effects for interactivity.

## 🛠️ Technologies Used

- **HTML5**: Structuring the content.
- **CSS3**: Custom styling, Flexbox & CSS Grid layouts, custom `@keyframes` animations, and responsive media queries.
- **JavaScript (Vanilla)**: Intersection observer logic to trigger CSS reveal classes on scroll.

## 🚀 How to Run the Project

1. Download or clone this repository to your local machine.
2. Navigate to the project folder.
3. Open `task1.html` directly in your favorite web browser (no local server or build tools required!).
